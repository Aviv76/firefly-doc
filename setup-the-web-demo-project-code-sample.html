Setup the Web Demo project code sample

You can follow each commit done in the preparation for this tutorial at: https://github.com/FireflyMigration/ENV.Web/tree/dev-demo-2017-10

Visual Studio Configuration

  • Visual studio 2017 version 15.3.5 or later
  • Make sure that both "ASP.NET and web development" and "Node.js development" options checked in Visual studio features configuration

Setup Typescript

  • Typescript sdk for Visual Studio

Setting up the SQL Server user

Open file WebDemo/Global.asax.cs

Locate the following line of code:

ConnectionManager.SetDefaultUserAndPassword("sa", "MASTERKEY");

Please change the user to any SQL user that has access to your database. Since this code runs under IIS, and not the current windows user, an SQL user is required.
